Examples
========

This page collects tutorial notebooks for BayesFlow, from simple starter examples
to more advanced modeling workflows. The corresponding Jupyter notebooks are
available :mainbranch:`here <examples/>`.

Tutorial notebooks
------------------

1. :doc:`Diffusion starter <notebooks/Diffusion_Models>` - A small tutorial on the power of diffusion models for SBI.
2. :doc:`Linear regression <notebooks/Linear_Regression_Starter>` - Fit your first Bayesian regression with varying sample size.
3. :doc:`Image data <notebooks/Spatial_Data_and_Parameters>` - Learn parameters from or generate image data.
4. :doc:`Bayes estimators <notebooks/Lotka_Volterra_Point_Estimation>` - From simple point estimates to fully Bayesian inference.
5. :doc:`Model comparison <notebooks/One_Sample_TTest>` - Learn Bayes factors using probabilistic classification.
6. :doc:`From ABC to BayesFlow <notebooks/From_ABC_to_BayesFlow>` - Upgrade from sequential to amortized inference.
7. :doc:`SIR <notebooks/SIR_Posterior_Estimation>` - Model infectious diseases through an end-to-end Bayesian workflow.
8. :doc:`Bayesian experimental design <notebooks/Bayesian_Experimental_Design>` - Perform adaptive sequential experiments.
9. :doc:`Estimating likelihoods <notebooks/Likelihood_Estimation>` - Learn synthetic likelihood functions.
10. :doc:`Multimodal data <notebooks/Multimodal_Data>` - Fuse different data types for more informative inference.
11. :doc:`Ensembles <notebooks/Ensembles>` - Train different networks at the same time and combine inferences.
12. :doc:`Ratio estimation <notebooks/Ratio_Estimation>` - Learn neural ratios for downstream MCMC sampling.

More tutorials are always welcome! Please consider making a pull request if you
have a cool application that you want to contribute.
